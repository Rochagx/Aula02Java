# Aula 03 - Fundamentos de Java

Este repositório contém exemplos práticos desenvolvidos durante a Aula 03 de Java.

## 📚 Conteúdos abordados

Durante esta aula foram estudados conceitos fundamentais da linguagem Java:

- Declaração de variáveis
- Tipos de dados (String, int, float, boolean)
- Atualização de valores de variáveis
- Operações matemáticas básicas
- Entrada de dados com Scanner
- Estruturas condicionais (if / else)

## 💻 Tecnologias utilizadas

- Java
- Scanner (entrada de dados)
